## Installation

Run boostrap script using the following command:

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/AnH0ang/dotfiles/main/bootstrap.sh)"
```

- neofetch
- nvim
- readline
- ssh
- starship
- tmux
- vscode

- zsh
